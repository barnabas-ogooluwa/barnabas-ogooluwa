# Advanced Induction Welding: Research & Portfolio

A high-fidelity, research-focused web project exploring **High-Frequency Induction (HFI) Welding** for seamed pipes and square tubes. This project serves as both a technical research repository and a professional student portfolio for **Barnabas Ogo-Oluwa**.

## 🏗️ Project Overview

This project investigates the metallurgical and electrical complexities of HFI welding—the backbone of modern infrastructure manufacturing. It covers:
*   **Process Physics:** Detailed analysis of Skin and Proximity effects in 150-450 kHz environments.
*   **System Components:** Deep dive into HF power supplies, induction coils, and ferrite impedors.
*   **Manufacturing Flow:** Step-by-step walkthrough from steel strip forming to final shaping.

## 🎨 Design Theme: "Industrial Tech"

The project uses a custom-crafted **Industrial Tech** aesthetic, designed to reflect the raw energy and precision of manufacturing:
*   **Palette:** Deep Charcoal (`#121417`) and Induction Glow Orange (`#ff6600`).
*   **Aesthetic:** Glassmorphism navigation, responsive 60/40 hero layouts, and high-contrast typography.
*   **Typography:** Modern 'Inter' font stack for clean, technical readability.

## 📁 Project Structure

*   `index.html`: The personal landing page featuring the author's portfolio and project introduction.
*   `research.html`: A high-level overview of the research assignment, key objectives, and manufacturing pillars.
*   `assignment.html`: The full technical research paper, including multimedia demonstrations and APA 7 references.
*   `style.css`: Centralized stylesheet containing the custom industrial design system.
*   `/assets`: Directory containing technical diagrams, profile placeholders, and process imagery.
